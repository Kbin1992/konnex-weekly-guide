# Konnex Weekly 500 KP Guide (Android)

> ✅ Works on Android via SubWallet app  
> ❌ Chrome extension has faucet errors — don't use it  
> ⚠️ Do everything inside the SubWallet browser, not Chrome

---

## Before You Start

- Android phone
- SubWallet app → search **"SubWallet"** on Google Play Store
- Twitter/X account

---

## One-Time Setup (Do This Once)

### 1. Install SubWallet
- Download from Google Play Store
- Create a new wallet **or** import with your existing seed phrase
- Save your seed phrase somewhere safe

---

### 2. Connect to Konnex Hub
1. Open SubWallet → tap **dApps** tab (bottom of screen)
2. Go to:
   ```
   hub.konnex.world/points?referral_code=SHTAUUZT
   ```
3. Tap **Connect** → choose **SubWallet** → select **Meta** account → tap **Connect**
4. Tap **Click to Sign** → approve in SubWallet
5. ✅ You're logged in

---

### 3. Go to Testnet Quest Portal
1. In SubWallet browser go to:
   ```
   subnets.testnet.konnex.world/quest
   ```
2. Tap **Verify Snag** tab
3. Tap **Connect EVM wallet** → select Meta → Connect
4. Tap **Verify Snag account** → approve signature
5. ✅ Your KNX wallet address is created automatically

---

### 4. Add KNX Testnet Network to SubWallet
1. In SubWallet → **Settings** → **Manage Networks** → tap **+**
2. Paste this RPC URL:
   ```
   wss://testnet-rpc1.konnex.world:39944
   ```
3. Fields auto-fill → tap **Save**
4. ✅ Konnex Testnet added

---

## Every Week (Repeat This for 500 KP)

### 5. Get Faucet Tokens
1. Go to quest portal → tap **Get Faucet** tab
2. Tap **Get testnet KNX**
3. ✅ Wait 1–2 mins for tokens to arrive

---

### 6. Submit Drone Request
1. Tap **Submit Request** tab
2. Pick any **Drone Location**:
   - Parking Area - open lot scan
   - Downtown Towers - dense urban route
   - River Corridor - linear shoreline pass
3. Tap **Next**
4. Pick any **Drone Task**:
   - Inspect - detailed visual check
   - Patrol - routine monitoring run
5. Tap **Next** → SubWallet shows **Signature Request**
6. Tap **Approve**
7. ✅ Blue **Done ✓** appears → transaction recorded on-chain

---

### 7. Claim Your 500 KP
1. Go to:
   ```
   hub.konnex.world/points
   ```
2. Scroll to **Product Tasks**
3. Find **Testnet Onchain Onboarding (Weekly)**
4. ✅ 500 KP added to your balance

> ⏰ Resets every week. Repeat Steps 5 → 6 → 7 each week.

## Common Issues

**Faucet not working?** → Make sure you completed Step 3 (Verify Snag) first

**SubWallet not showing as Installed?** → You must be inside SubWallet's dApps browser, not Chrome

**Points not showing?** → Can take up to 24 hours to update

**Signature not popping up?** → Switch to SubWallet app manually after tapping the button
